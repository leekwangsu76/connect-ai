# ⚙️ 워크플로우: Antigravity + Connect AI 협업 체계

## 🔑 핵심 원칙
> **Antigravity = 감독 (토큰 유료, 최소 사용)**
> **Connect AI = 선수 (Ollama → Gemma 4 E4B, 100% 로컬, 토큰 무료)**

Antigravity는 전략 설계 + 브리프 작성만 하고,
실제 조사/글쓰기/분석은 Connect AI 에이전트가 무료로 수행한다.

---

## 📋 작업 흐름

```
[1] 사용자 → Antigravity에게 말하기
    "이번에 홍대 맛집 비교해 줘"
    (한 줄이면 충분)

[2] Antigravity → 명령 텍스트 작성
    - Connect AI 채팅에 붙여넣을 구체적인 명령문을 작성
    - 대상, 수량, 형식을 명확히 포함
    - ⚠️ "제안서 금지, 결과물만 제출" 원칙 포함

[3] 사용자 → Connect AI 채팅창에 복사 붙여넣기
    - Antigravity가 써준 명령 텍스트를 그대로 복붙
    - 🧭 CEO가 자동으로 수신 → 작업 분해 → 에이전트에게 분배
    - 각 에이전트가 자기 역할에 맞게 실행

[4] Connect AI 에이전트 → 산출물 자동 생성
    - sessions/ 폴더에 자동 저장 (d:\connect-ai\lkshub\sessions\)

[5] 사용자 → Antigravity에게 검토 요청 (선택)
    "결과 나왔는데 검토해 줘"
    - Antigravity가 sessions/ 폴더의 산출물 품질 확인
    - 부족하면 보완 명령 텍스트 추가 작성
```

## 🤖 Antigravity의 역할 (이것만 한다)

|---|---|---|
| 🧠 전략 설계 | 사용자의 한 줄 요청을 구체적 작업으로 분해 | 소량 |
| 📝 브리프 작성 | Connect AI 에이전트에게 보낼 명령서 작성 | 소량 |

### Antigravity가 하지 않는 것 (토큰 절약)
- ❌ 직접 웹 검색해서 데이터 수집 (→ Connect AI Researcher가 함)
- ❌ 직접 스크립트/카피 작성 (→ Connect AI Writer가 함)
- ❌ 직접 영상 기획안 작성 (→ Connect AI YouTube가 함)
- ❌ 직접 코드 개발 (→ Connect AI Developer가 함)

## 📏 브리프 작성 규칙 (Antigravity가 지킬 것)

### 반드시 포함할 5가지
1. **대상** — "어디서, 무엇을" (예: 성수동 카페 3곳)
2. **수량** — "몇 개, 몇 분 분량" (예: 비교표 1장, 스크립트 10분)
3. **형식** — "어떤 포맷으로" (예: 표, 스크립트, 슬라이드)
4. **금지** — "전략 제안서 금지, 결과물만 제출"
5. **에이전트별 배분** — 누가 뭘 하는지 명확히

### 절대 하지 말 것
- ❌ "~에 대해 분석하세요" (너무 추상적)
- ❌ "전략을 수립하세요" (또 보고서만 나옴)
- ❌ "자료를 정리하세요" (뭘 정리하라는 건지 불명확)
